# DiscordCTFBot

Discord bot to manage ongoing CTF

## Install

**Python 3.8 or higher is required**

```sh
$ git clone https://github.com/zVaz/DiscordCTFBot.git
$ cd DiscordCTFBot
$ pip3 install -r requirements.txt
```

Create `.env` file:
   ```
   DISCORD_TOKEN=<discord token>
   GUILD_ID=<guild id>
   ```

## Run

```sh
$ python3 bot.py
```
